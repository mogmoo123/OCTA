======HOW TO USE IT======
- OCTA || 자동 알림 프로그램
1. win + R을 열고, "shell:startup" 이라고 적은 후 엔터
2. OCTA 파일을 그곳에 넣는다.
3. ORCA 계산을 돌려서 잘 작동하는지 확인한다.

======CUSTOMIZE======
-OCTA || 자동 알림 프로그램
1. 원하는 아이콘, 음성파일을 src파일에 넣는다.
2. SETTINGS.json 파일에 PYTHON 상대 경로로 각 파일을 집어 넣는다.
3. (이미 사용중인 경우) 컴퓨터를 껐다 킨다. / (첫 설치 후) 프로그램 실행
4. ORCA 계산 돌려서 확인해본다.
