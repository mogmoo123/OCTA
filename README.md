# OCTA
Project RUKA, 그 첫 단계, ORCA Completement Toast Alarm (OCTA) System

# 배포판
- [Install Link](https://drive.google.com/file/d/1PH-zzck9f0qf5Bv4eieKU5W8z65QFiT4/view?usp=sharing) for Window

# 사용법
1. win + R을 열고, "shell:startup" 이라고 적은 후 엔터하여 시작프로그램 파일을 연다.
2. 다운로드 한 파일 속 "OCTA" 라는 이름의 파일 (.exe 아님) 을 시작프로그램으로 옮긴다.
3. 옮긴 바로가기를 열어서 프로그램을 시작한다.
4. ORCA 계산을 돌려서 잘 작동하는지 확인한다.

# 커마 하는 법

## 아이콘, 알람음 추가
1. 원하는 아이콘, 음성파일을 src파일에 넣는다.
2. SETTINGS.json 파일에 PYTHON 형식 상대 경로로 각 파일의 경로를 입력한다.
 ex) "icon" : "src/example.ico"  
3. 파일 저장 후 프로그램을 재시작하여 적용한다.
   
## 아이콘, 알람음 제거
1. SETTINGS.json을 연다.
2. 파일 경로가 적혀있는 부분을 ""으로 바꾼다.
3. 파일 저장 후 프로그램을 재시작하여 적용한다.
