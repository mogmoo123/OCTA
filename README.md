# OCTA
Project RUKA, 그 첫 단계, ORCA Completement Toast Alarm (OCTA) System

# 배포판
- [Install Link](https://drive.google.com/file/d/1PH-zzck9f0qf5Bv4eieKU5W8z65QFiT4/view?usp=sharing) for Window

======HOW TO USE IT======
- OCTA || 자동 알림 프로그램
1. win + R을 열고, "shell:startup" 이라고 적은 후 엔터
2. OCTA 파일(파일 이름이 OCTA이고, 바로가기인 파일)을 그곳에 넣는다.
3. ORCA 계산을 돌려서 잘 작동하는지 확인한다.

======CUSTOMIZE======
1. 원하는 아이콘, 음성파일을 src파일에 넣는다.
2. SETTINGS.json 파일에 PYTHON 상대 경로로 각 파일을 집어 넣는다.
3. (이미 사용중인 경우) 컴퓨터를 껐다 킨다. / (첫 설치 후) 프로그램 실행
4. ORCA 계산 돌려서 확인해본다.

+) 음성 제거를 원할 경우 "None"으로, 아이콘 제거를 원할 경우 ""로 두기
